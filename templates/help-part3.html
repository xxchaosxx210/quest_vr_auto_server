Known Issues:

Below are some known issues that Users have encountered. Hopefully this will help with any issues installing or running QuestCave

Issue: When I install and try to run QuestCave my Anti Virus Blocks it.

Solution: This is a false positive. QuestCave is a Windows application and is not a virus. You can add QuestCave to your Anti Virus Exclusion list or disable your Anti Virus while you are using QuestCave. Below is an example of Microsoft Security alert requesting to allow QuestCave to run. Simply click on the 'more info' link and then click on the 'Run Anyway' button.
